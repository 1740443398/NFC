🚀 ESP32-S3 多功能 NFC 语音快捷启动器
 
ESP32-S3-N16R8 + PN532 + I2S 麦克风 + OLED 四按键屏幕
NFC 写入 · 语音控制 · Web 后台配置 · 系统日志
 ✨ 功能特性
 
- 🔑 NFC 标签写入 — 4 个按键可绑定 APP、网址、自定义指令
- 🎤 口语化语音控制 — 长按语音键，直接设置按键功能
- 📶 WiFi 网页后台 — 浏览器修改配置、查看系统运行日志
- 📺 OLED 实时显示 — 按键列表、WiFi IP、录音状态界面
- 🔋 低功耗关机 — 长按按键进入深度休眠
- 🎵 蜂鸣器提示音 — 操作状态实时反馈

 🧰 硬件清单
 - ESP32-S3-N16R8 开发板
- PN532 NFC 模块
- I2S 麦克风模块
- 4 按键 I2C OLED 屏幕
- 无源蜂鸣器
- 杜邦线 / 焊接线材

 🎮 按键功能
 
- K1 / 按键1：短按 → 写入 NFC；长按 → 关机
- K2 / 按键2：短按 → 写入 NFC；长按 → 显示 WiFi IP
- K3 / 按键3：长按 → 录音语音识别；松开 → 解析指令
- K4 / 按键4：短按 → 写入 NFC

 ⚠️ 重要说明 · 门禁卡相关
 
关于加密门禁卡
 
本项目仅支持普通 NFC 标签写入功能，不支持、也无法破解、复制加密门禁卡。
 
- 小区/公司等加密 M1 卡、CPU 卡、带动态密钥的门禁卡均无法复制
- PN532 不具备解密、嗅探、破解加密卡片的能力
- 代码中无任何破解、攻击、暴力破解逻辑

 🚀 使用方法
 
使用方法
 
1. 按照接线表完成硬件焊接或直插连接
2. 在 Arduino IDE 中安装 ESP32-S3 核心，并在库管理器安装以下必需库：
- U8g2 （OLED 屏幕驱动库）
- PN532 （NFC 模块驱动库）
> [!NOTE]
> 如果使用 米思齐（Mixly），不需要额外安装上面两个库，软件已自带，直接选择 ESP32-S3 芯片即可使用。
3. 使用 Arduino IDE（需安装 ESP32-S3 核心）或PlatformIO 编译并烧录 main.cpp
4. 上电后 OLED 自动显示按键列表
5. 长按按键 2 查看 WiFi IP，浏览器访问进入后台
6. 短按任意按键进入 NFC 写入模式
7. 长按按键 3 启动语音设置功能。
